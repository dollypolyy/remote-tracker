# remote tracker 🌸

личный трекер для поиска удалённого заработка.

## запуск

```bash
npm install
npm run dev
```

открой `http://localhost:5173` в Chrome или Яндекс.Браузере — там работает голосовой ввод.

## разделы

- **способы** — карта идей заработка с оценками
- **люди** — контакты и статус общения
- **гипотезы** — конкретные тесты с метриками
- **итог дня** — дневник + статистика + ежедневный чеклист

## данные

всё хранится в `localStorage` браузера. используй кнопки **экспорт json** / **импорт json** в шапке, чтобы делать резервные копии или переносить данные.

## голосовой ввод

работает через Web Speech API в Chrome / Яндекс.Браузере на `localhost`. кнопка 🎙 есть у каждого текстового поля.
